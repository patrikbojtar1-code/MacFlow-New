## ✨ About

**MacFlow New** is an enhanced version of the original NotchLand concept.

Instead of simply copying the original project, we used it as a foundation and redesigned multiple parts of the application by adding new functionality, improving performance, refining the UI, and restructuring the codebase.

Our goal is to provide a smoother, faster, and more customizable experience for macOS users.

---

## 🚀 Features

* Modern Dynamic Island / Notch integration
* Improved animations
* Better performance and responsiveness
* Cleaner UI
* Bug fixes and stability improvements
* Optimized code structure
* Easier customization
* Active development
